## Шаги

#### Функциональность фильтра

Настройте функцию фильтрации, при которой, как только пользователь нажимает на кнопку, отображаются только те пункты меню, которые относятся к выбранной категории. Для этого определите функцию, которая принимает категорию в качестве параметра и обновляет состояние, чтобы отображать только те пункты меню, которые относятся к этой категории. Затем вы можете передать эту функцию компоненту Categories в качестве вспомогательного средства и прикрепить ее к кнопкам.

Когда пользователь нажимает на кнопку категории, должна вызываться функция фильтрации с выбранной категорией в качестве параметра. Затем функция должна обновить состояние, чтобы отображать только те пункты меню, которые относятся к выбранной категории.

В целом, работа приложения должна выглядеть примерно так:

- Настроить массив категорий в качестве переменной состояния в компоненте App.jsx.
- Определите функцию, которая принимает категорию в качестве параметра и обновляет состояние, чтобы отображать только те пункты меню, которые относятся к этой категории.
- Подключите функцию фильтрации к кнопкам категории в компоненте Категории.
- Повторяйте шаги 9-10, пока пользователь не выберет другую категорию или не решит выйти из компонента меню.

#### Установить объект

[JS Nuggets - новый набор()](https://www.youtube.com/watch?v=H4NnCItCZWE&list=PLnHJACx3NwAfRUcuKaYhZ6T5NRIpzgNGJ&index=26)

В JavaScript объект Set представляет собой набор уникальных значений. Он позволяет хранить значения любого типа, такие как примитивные типы (числа, строки, логические значения) и ссылки на объекты.

Вот простой пример использования Set:

```js
// Create a new set
let mySet = new Set();

// Add values to the set
mySet.add(1);
mySet.add(2);
mySet.add(3);

// Add a duplicate value (ignored)
mySet.add(1);

// Get the size of the set (3)
console.log(mySet.size);

// Check if a value is in the set (true)
console.log(mySet.has(2));

// Remove a value from the set
mySet.delete(2);

// Get an array of all values in the set
let myArray = Array.from(mySet);
console.log(myArray); // [1, 3]
```

const tempCategories = menu.map((item) => item.category);
