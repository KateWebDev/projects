## Шаги

#### Основная логика

Чтобы перемещать слайды вперед и назад, используйте свойство transform со значением translateX. Например, чтобы переместить слайды влево, вы можете использовать свойство transform:translateX(-100%), однако для перемещения слайдов вправо вы можете использовать свойство transform:translateX(100%). Для активного слайда мы будем использовать transform:translateX(0)

#### Текущий пользователь

Создайте значение состояния текущего пользователя в App.jsx и изначально установите его равным 0. Это позволит вам отслеживать отображение текущего слайда.

#### Предыдущий и следующий

Реализуйте функции prev и next, используя функцию setCurrentPerson для обновления значения состояния текущего пользователя.

#### Автоматическое переключение

Реализуйте функцию автоматического переключения, используя функцию setInterval для изменения значения состояния текущего пользователя через регулярные промежутки времени.

#### Библиотека Extra - React Slick

[React Slick Docs](https://react-slick.neostack.com/)

```sh
npm install react-slick slick-carousel --save
```

В целом, работа приложения должна выглядеть примерно так:

- В App.jsx импортируйте массив people из data.js и установите его в качестве значения состояния, используя useState hook.
- В инструкции return настройте элемент container для хранения всех слайдов и выполните итерацию по значению people state для создания каждого слайда.
- Настройте кнопки prev и next и оформите контейнер, слайды и кнопки.
- Раскомментируйте остальные элементы в массиве people.
- Создайте значение состояния текущего пользователя в App.jsx и установите его равным изначально.
- Реализуйте функциональность prev и next, используя функцию setCurrentPerson для обновления значения состояния currentPerson и перемещения слайдов вперед и назад, используя свойство transform со значением translateX.
- Реализуйте функцию автоматического перемещения с помощью функции setInterval для изменения текущего значения состояния пользователя через регулярные промежутки времени.
