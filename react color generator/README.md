## Steps
 
```

#### Визуализация цветов

Передайте список цветов компоненту colorList.

В компоненте colorList выполните итерацию по списку цветов и визуализируйте каждый из них в одноцветном компоненте. Устраните ключевую проблему, указав уникальный идентификатор для каждого цвета.

В компоненте SingleColor отобразите шестнадцатеричное значение и вес (в процентах) каждого цвета и используйте встроенный CSS для настройки цвета фона компонента.

#### React-Toastify

Импортируйте и настройте библиотеку react-toastify.

[React Toastify](https://fkhadra.github.io/react-toastify/introduction)

```sh
npm i react-toastify
```

main.jsx

```js
import 'react-toastify/dist/ReactToastify.css';
```

App.jsx

```js
import { ToastContainer, toast } from 'react-toastify';

toast.success('awesome');
toast.error('error message');

return (
  <main>
    ...............
    <ToastContainer position='top-center' />
  </main>
);
```

#### Создайте цвет

В компоненте приложения создайте логику для создания нового списка цветов, когда пользователь вводит новое значение цвета. Используйте react-toastify для отображения сообщения об ошибке, если библиотеке values.js не удается сгенерировать цвет, а также для отображения всплывающего сообщения, если пользователь пытается ввести пустое значение.

#### Буфер обмена

В компоненте SingleColor настройте обработчик щелчка, который сохраняет шестнадцатеричное значение цвета в буфер обмена браузера.

Чтобы сохранить какое-либо значение в буфере обмена navigator, вы можете использовать API Clipboard, который является частью веб-API, предоставляемого современными браузерами. API Clipboard позволяет веб-разработчикам взаимодействовать с буфером обмена пользователя, включая чтение и запись в него данных.

Вот пример того, как сохранить строковое значение в буфер обмена с помощью Clipboard API:

```js
async function copyToClipboard(text) {
  try {
    await navigator.clipboard.writeText(text);
    console.log('Text copied to clipboard');
  } catch (error) {
    console.error('Failed to copy text: ', error);
  }
}

const textToCopy = 'Hello, world!';
copyToClipboard(textToCopy);
```
